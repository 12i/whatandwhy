这个脚本主要面向国人，并且会随时更新。它会安装大部分你在termux中需要的东西

# 安装方法
```
pkg install git
git clone https://github.com/nidhoggfgg/termux-tools-install.git tools
cd tools
bash tools.sh
```
然后你就可以安装好它们了，加入了个性化，将每个工具都分了，加入多线程，更快更高效!
